# AGENTS.md Example

## Scope

- Work only within the current repository unless explicitly told otherwise.

## Allowed Without Prompt

- Read files and list directories
- Implement changes within the approved scope
- Run safe validation commands

## Ask First

- Install packages
- Delete files
- Push changes
- Access secrets or external systems
- Run destructive operations

## Delivery Rules

- Keep diffs focused and small
- Log meaningful steps in `PROGRESS.md`
- Report validation results before marking work complete
