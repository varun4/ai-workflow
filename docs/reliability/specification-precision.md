# Specification Precision

Specification precision is the discipline of turning vague intent into
clear, bounded, testable work.

## What Precision Looks Like

- The task is concrete
- Constraints are explicit
- Success is measurable
- Non-goals are visible

## Weak Example

"Improve this workflow using AI."

## Stronger Example

"Create a course repo scaffold for developers and knowledge workers
with shared docs, split templates, and an initial module structure. Do
not duplicate the core docs by audience."

## Common Failure Patterns

- Ambiguous goal
- Hidden assumptions
- No definition of done
- Scope too broad for one execution step

## Teaching Exercise

Give learners three vague requests and ask them to rewrite each as an
executable spec.
