# Principles

## Structure Beats Prompting

Reliable agentic work comes more from workflow design than from clever one-off prompts.

## Bounded Autonomy Beats Vague Delegation

Agents perform better when the task, tools, scope, and stopping conditions are explicit.

## Artifacts Beat Chat History

Durable files such as specs, progress logs, templates, and checklists
create continuity that a chat transcript cannot.

## Evaluation Is Part Of The Work

Output without verification creates hidden rework and false confidence.

## Security Must Be Designed In

Permission boundaries, secret handling, and approval gates should be
part of the workflow from the start.

## Examples Should Be Transferable

The course should teach patterns that apply across tools, teams, and levels of automation.
