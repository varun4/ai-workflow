# Module 01 Facilitator Notes

## Teaching Emphasis

- Keep the opening practical, not abstract
- Use examples from real work, not hypothetical hype
- Reframe bad outcomes as workflow failures instead of model failures

## Common Student Mistakes

- Focusing on tools before defining the workflow
- Asking for automation before defining boundaries
- Treating prompts as a substitute for process design

## Discussion Prompt

Ask learners to name a recent task where AI produced something
plausible but wrong, then identify which artifact was missing.
