# Developer Track Guide

Use the shared docs as the main framework, then translate each concept
into software delivery workflows.

## Focus Areas

- feature specs
- bugfix scope
- code review workflows
- test and validation loops
- repository-level operating rules

## Most Relevant Templates

- `templates/shared/AGENTS.example.md`
- `templates/shared/PROGRESS.example.md`
- `templates/shared/SPEC.example.md`

## Capstone Direction

Design an agent-assisted software delivery workflow for a small feature
from spec through validation.
