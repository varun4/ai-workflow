# Example AGENTS.md (Knowledge-Work Workflow)

## Scope

- Work only on vendor comparison and recommendation brief production.
- Limit output to approved decision criteria and evidence sources.

## Allowed Without Prompt

- Review approved internal source materials.
- Summarize and compare vendor capabilities in scoped criteria.
- Produce draft recommendation artifacts.
- Run approved quality checks.

## Ask First

- Including non-approved external sources.
- Adding legal, procurement, or security claims not in evidence.
- Sharing draft outside approved stakeholder group.
- Final distribution to leadership.

## Delivery Rules

- Keep claims traceable to cited sources.
- Separate facts, assumptions, and recommendations.
- Record progress and review checkpoints in `PROGRESS.md`.
