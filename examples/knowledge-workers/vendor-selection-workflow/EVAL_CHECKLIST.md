# Evaluation Checklist

## Correctness

- Facts match cited sources.
- Scoring values align with documented criteria.

## Completeness

- All vendors are evaluated across all required criteria.
- Tradeoffs and risks are explicitly documented.

## Constraint Compliance

- Only approved sources are used.
- Confidential details follow sharing boundaries.

## Quality

- Recommendation is clear, concise, and defensible.
- Evidence trail is easy for reviewers to audit.

## Approval

- Stakeholder review completed.
- Leadership distribution approval recorded.

## Decomposition Quality

- Each phase has bounded tasks.
- Each phase has one concrete output.
- Each phase has a checkpoint decision.
- Escalation conditions are explicit at approval gates.
