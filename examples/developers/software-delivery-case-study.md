# Developer Case Study

## Scenario

Build a small feature using an agent-assisted workflow with explicit
rules, progress logging, and evaluation.

## Artifacts To Show

- feature spec
- repository `AGENTS.md`
- `PROGRESS.md` entries
- skill usage
- evaluation checklist

- `examples/developers/rbac-admin-workflow/AGENTS.md`

- `examples/developers/rbac-admin-workflow/APPROVAL_BOUNDARIES.md`

## Teaching Goal

Show how the workflow reduces ambiguity, improves reviewability, and
makes agent output easier to trust.
