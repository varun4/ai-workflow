# Skills Index

Skills are reusable instruction packages that make agent behavior more
consistent across repeated tasks.

## Purpose

- Reduce repeated prompting
- Capture proven execution patterns
- Standardize review and evaluation behavior
- Make workflows easier to teach and reuse

## Initial Skill Categories

- Spec writing
- Task decomposition
- Context packaging
- Evaluation and review
- Security checks

## Starter Skills To Add

1. `spec-writer.md`
2. `task-decomposer.md`
3. `context-packager.md`
4. `evaluator.md`
5. `security-reviewer.md`

## Skill Design Rules

- Keep each skill focused on one repeatable capability
- Prefer checklists and decision rules over long prose
- Include inputs, outputs, boundaries, and verification steps
- Update skills when a repeated failure pattern becomes visible
